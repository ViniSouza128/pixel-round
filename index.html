Info

A pixel-perfect circle, ellipse, sphere and ellipsoid generator.

1. What it is

Browser-based generator for pixel-art circles & ellipses (2D) and voxel spheres & ellipsoids (3D). Integer dimensions in, PNG out. No installation, no account, no backend.

2. Modes & shapes

2D · Circle

Single Size. Filled, thin or thick pixel circle.

2D · Ellipse

Width + Height independent. Same render-mode set.

3D · Sphere

Single Size. Isometric voxel sphere. Cut on X or Y slices through.

3D · Ellipsoid

Width + Height + Depth independent. Cut slices on the chosen axis.

3. Algorithms (2D only)

Euclidean

Distance test at pixel centres. Smoothest contour.

Bresenham

Classic integer midpoint algorithm. Stair-stepped pixel-art look.

Threshold

Corner-coverage test. Chunkier silhouette — any cell with a corner inside fills.

4. Controls

Mode & shape

The 2D / 3D and Circle / Ellipse (or Sphere / Ellipsoid in 3D) toggles at the top are the ONLY way to switch.

Cut (3D)

The X / Y toggle chooses the slicing axis. Switching axis restores the full figure automatically — only one axis cuts at a time. The slider max equals the chosen axis dimension.

Pinch & rotate

Two fingers on the canvas zoom. In 3D, the midpoint of the fingers also rotates the figure. Mouse wheel zooms in 3D; click-drag rotates.

Grid & wireframe

The grid button in the canvas top-left toggles the cell grid in 2D and wireframe in 3D.

Keyboard

G Grid   C Center   D Download   T Theme   I Info chip   M 2D/3D   S Sound

5. Technical

Stack

Vanilla JavaScript + Canvas 2D + three.js. No framework, no build step.

Offline

PWA-installable. Once loaded, no network is required. Settings persist via localStorage.

Output

PNG with the accent color on transparent background. 2D exports at ~2048px on the major axis.
